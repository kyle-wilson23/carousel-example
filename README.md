# carousel-example

This is an example carousel component that displays a list of videos. Only the active video can be interacted with (one video plays at a time). Video playback controls are available on hover (always available on mobile). Responsive - supports all devices.

## Tech Stack

* Typescript
* React
* Swiper.js for the carousel implementation

## Usage

* videos can be found [here](https://drive.google.com/drive/folders/1brniOB6XXsoj2v1jIXxEpUq0nV8WwjQF?usp=drive_link)
  * alternatively, any collection of videos with filenames `i.mp4` where `i` is some integer and the files are organized in numerical order should be supported. Must also be portrait orientation
* drop the videos in the `/public/assets/videos` directory.
* `npm install`
* `npm run dev`
